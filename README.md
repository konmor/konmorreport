# KonmorReport

## 介绍

用于创建各类可视化图表 包含 柱状图、条形图、折线图、饼图、散点图、雷达图、仪表、标签、表格。其中包含各种变种如
柱状图中可设置为折线图，饼图可调整为环形图、南丁格尔图，仪表可调整为进度条样式，表格可根据数据呈现柱状进度条或电池条等等，等你探索。

## 开源初衷

让二维数据可视化变得简单、易用、好用
